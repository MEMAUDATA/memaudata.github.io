Data science projects and not

Hearing data

Flask web app for online hearing clinic database analysis. The analysis is able to predict the number of customers for the next year.

Tools used : Flask - Pandas - Numpy - scikit-Learn - prophet

This app is used for Enfants sourds du Cambodge NGO data analysis.

MEMAU

Web platform dedicated to audiology, primiraly designed to help students for their final years project. Core functionnalities : ressources librairy, questionnaires, tools for statistics and bibliography.Open-source.

Developped under Wordpress and Divi.

memau.eu

Audiodetect

Audiogram detection and data storage using the webcam. On-going project.

Tools used : Pytorch - Pandas.

MEMAUDATA

Web plateform to analyze online audiological questionnaires. Open-source. On-going project.

Tools used : WordPress - Gravity form.
